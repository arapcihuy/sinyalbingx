# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly via:

- Email: rasyid@uii.ac.id
- Subject: [SECURITY] sinyalbingx vulnerability report

Please include:
- Affected component
- Steps to reproduce
- Impact assessment
- Suggested remediation, if any

I will review valid reports as soon as possible.

## Scope
This repository is intended for authorized research, development, and deployment workflows only.
